API Documentation

Complete reference for all endpoints

Movies API

GET /movie/search?q={query}
Search movies on CineSubz
Example: GET /movie/search?q=oppenheimer
GET /movie/info?url={movie_url}
Get complete movie details with download links
Example: GET /movie/info?url=https://cinesubz.net/movies/oppenheimer-2023/
GET /movie/recent
Get recently added movies
GET /movie/popular
Get popular/trending movies

Games API

GET /game/fitgirl-search?q={query}
Search games on FitGirl Repacks
Example: GET /game/fitgirl-search?q=cyberpunk
GET /game/fitgirl-info?url={game_url}
Get complete game info with repack features and download links
GET /game/fitgirl-download?url={fuckingfast_url}
Extract direct download link from fuckingfast.co

Anime API

GET /anime/search?q={query}
Multi-source anime search (AniList + Jikan)
Example: GET /anime/search?q=naruto
GET /anime/popular
Get top 20 popular anime
GET /anime/info?id={anime_id}
Get detailed anime information by MAL ID

All APIs Aggregator

GET /all-apis
Get all API data in one response
GET /all-apis?section={section}
Get specific section (game/movie/anime)
GET /all-apis/movie?action=search&q={query}
Search movies through aggregator
